Detailed Description

Defines constructors for immutable planner action records.

Action Records are the planner output described by ADR-047. They represent what the bootstrap engine intends to do without encoding how that work will be performed on a particular operating system or package manager.

The current implementation serializes action records as pipe-delimited text so empty fields are preserved safely when Bash reads the record back. This encoding is an implementation detail rather than the architectural contract. Callers should create records through the constructor functions in this module instead of assembling records by hand.
